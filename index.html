Schieberegister

  • Viele Ausgänge aus wenigen Eingängen erzeugen
  • Ansteuerung seriell
  • Werte werden zwischgespeichert
  • Aktivierung übernimmt Werte auf die Ausgänge

Funktion

  1. Chip Enabled LOW
  2. Werte schreiben (je Farbe 8 Bit)
  3. Zielregister wählen (8 Bit)
  4. Chip Enabled HIGH
  5. Wartezeit für LED (2 ms)

Arduino / ESP 8266

static uint8_t data[4] = {0x0, 0x0, 0x0, 0x0};
int shift;   int displaywait = 2;
uint8_t pic[8] = {0xFF, 0x42, 0x24, 0x18, 0x18, 0x24, 0x42, 0xFF};
for (shift = 0; shift < 8; shift++)
{
    data[0] = ~pic[j];
    data[2] = 0xFF;
    data[1] = 0xFF;
    data[3] = 0x01 << shift ;
    digitalWrite(CE, LOW);
    SPI.transfer(data[0]);    SPI.transfer(data[1]);
    SPI.transfer(data[2]);    SPI.transfer(data[3]);
    digitalWrite(CE, HIGH);
    delay(displaywait);
}

Additive Farbmischung

ROT + GRÜN = GELB
GRÜN + BLAU = CYAN
ROT + BLAU = MAGENTA
ROT + GRÜN+ BLAU = WEIß

Wichtig: Mehr als eine LED je Feld kann nicht aktiviert werden!

Die Mischung erfolgt über ein zeitliches Multiplexing.
